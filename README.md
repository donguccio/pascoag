Pascoag
=======

Pascoag is a Mac OS X application that provides text-terminal access to CollectiveAccess databases using the VT-52 or <a href="http://www.vt100.net">VT-100</a> protocols.
Why would you ever want to do this? For fun. To pretend you're living in 1986. Or to <a href="http://lost.deaccession.org">make a point</a> I suppose.
